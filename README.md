# 羽扬日记

A private couple diary PWA.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- PostCSS

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Routes

- `/` → redirects to `/home`
- `/home` — today's entry
- `/chapters` — chapters
- `/memories` — memories
- `/us` — about us / settings
